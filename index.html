Futuro App
Aplicación movil y backend para partido político "Futuro"

Desarrollo de App para el partido político "Futuro".
Es una aplicación móvil (SO Android), para registrar voluntarios defensores de casillas de votación.
Mediante un formulario se recaba información personal del voluntario, se manda a una base de datos en un servidor privado, para posteriormente ser contactados por un integrante de dicho partido político.
La app fue desarrollada en Java, MySQL, el servidor en framework Express.
